目的：网页正文内容抽取

约定：
	所有的网页是由两部分组成的：<a>标签与非<a>标签
	通常，广告，导航 这些非正文内容，使用<a>的频率较高
	有些网页包括在标签里面的css 或 javascript 较多，有的则较少
	有些网页会经过内容的压缩，这里压缩是指去掉所有对显示没有影响的空格和换行
	

steps:
1 将整个html 网页去掉所有的<script>... </script>   <style>....</style> 的内容，并且删除所有的换行，使之只剩下一行。
2 将上面的内容.split('</a>')  分成多行
3 for 每一行计算三个值：    每行的HTML标签的个数 t1；   每行中<a> 里面的文本的长度t2；    每行所有文本的长度t3       (文本：删除html标签的文本）
4 计算每一行的值    (t2*w2 + t3*w3)/ t1            w2=0.3   w3=0.5

所在的html网页是由<a>标签与非<a>的标签组成的； 
我给<a>标签里面的文本赋予较低的权重；    非<a>的较高。
