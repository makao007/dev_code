    品铂的又一台10.1寸FHD平板M8HD开卖了。提到这个参数，大家可能会想起之前的M9 pro，这两款产品在硬件参数上都一致，区别在于M9 pro是塑料机身、背面和iPad相似，M8HD是金属机身、弧形背面(品铂自家称之为水滴状的边沿设计、书卷式造型)。
    M8HD的机身设计感觉挺不错的，延续了M8 pro的外观，能拿来类比的只有索尼Xperia Tablet Z。有网友提到蓝魔W41、W42，小编去了解了下，W41、W42虽然是金属机身，但四边边框厚度相当，不像M8HD上边厚下边薄。M8HD这种设计竖着握手感很不错。
蓝魔W41
    M8HD目前已经在淘宝内上架，官方参考价为Wi-Fi版16GB/1299元，3G版16G/1499元。目前还没看到有32GB版的，小编猜是为了和M9 pro区分开来吧，毕竟M9 pro可只有32GB版，两者在外观、存储空间上的区分，更容易让大家识别。
