<br />
    品铂的又一台10.1寸FHD平板M8HD开卖了。提到这个参数，大家可能会想起之前的M9 pro，这两款产品在硬件参数上都一致，区别在于M9 pro是塑料机身、背面和iPad相似，M8HD是金属机身、弧形背面(品铂自家称之为水滴状的边沿设计、书卷式造型)。<br />
<br />
    M8HD的机身设计感觉挺不错的，延续了M8 pro的外观，能拿来类比的只有索尼Xperia Tablet Z。有网友提到蓝魔W41、W42，小编去了解了下，W41、W42虽然是金属机身，但<a href="http://www.1pad.cn/2013/0115/9241.html">四边边框厚度相当</a>，不像M8HD上边厚下边薄。M8HD这种设计竖着握手感很不错。<br />
