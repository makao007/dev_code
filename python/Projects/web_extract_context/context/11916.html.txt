    之前九月末曝光的原道M8开始正式预售了，RK3188四核7.9寸1024*768屏幕1GB内存16GB存储空间，惯见的配置，售价是799元。
    看到这款产品，大家肯定会想起原道5月份推出的mini one和mini s，两款产品同样是7.9寸的迷你平板，除了内存在参数几乎一模一样。好吧，其实mini s的配置和M8确实是一模一样的(mini s现在京东售价699元)。那究竟M8有什么不同呢？
    小编对比了大半天，终于发现：M8的厚度是6.5mm，mini one、mini 上的厚度是5.9mm。这应该是外观上最显著的差别了，另外M8的摄像头从右侧移到了正中间，这不得不说是个大的改进，至少视频时不担心只显示一半脸咯。
    另外据小编观察，原道近半年来貌似只有4款产品，而且都是7.9寸的迷你产品，mini one、mini s、mini 3G、M8。这样的产品节奏在现在快推进的环境中有些格格不入，可能跟原道半年来公司人事变动不断有关吧。据传最近原道某部门的老大出走去单干了，小编希望公司在售后方面能够有所保证。
