全志科技刚刚预告了它的八核处理器A80 Octa。big.LITTLE架构让大家不由浮想联翩，这颗处理器绝对不是MT6592那样，只是性能孱弱的Cortex-A7架构，但它的运作方式还没有消息。<br />
big.LITTLE架构一共有三种运作模式，以A15+A7八核为例。一是Cluster Migration，只能运行4颗A7核心或运行4颗A15核心，这种就是三星常用的方式，效率低下为人诟病。二是CPU Migration，一颗A7和一颗A15搭配成一组，总共4组，每组只能有一颗运转，运转组数不限制，这样可以搭配从单核A7到四核A15一共14种变化，效率上好很多，但目前还没有厂商实现。<br />
