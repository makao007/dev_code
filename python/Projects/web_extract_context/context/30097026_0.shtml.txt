调查显示，韩国教师地位在包括经济合作与发展组织（OECD）会员国在内的21个主要国家中，排名较高，位居第4位。但经统计，在针对学生是否尊敬教师这一项进行调查的结果显示，在所有调查对象国中，韩国排名垫底。
10月5日（当地时间），国际教育机构巴乐基GEMS财团发布了综合教师年薪、社会地位等调查项目在内的&ldquo;教师地位指数（Teacher Status Index）&rdquo;报告书。该报告书显示，韩国总分为62分，位居中国（100分）、希腊（73.7分）、土耳其（68分）之后，排名第4位。教师地位指数是为了了解学生们的学业完成能力与教师地位、年薪的相关关系，由GEMS财团和英国苏塞克斯大学彼得&middot;道尔顿教授共同研究开发的指数。以美国、中国、英国等21个国家的不同职业、性别、年龄的1000名对象为标本进行调查，并以此为基础制作报告书。
根据该指数，韩国教师的人均年薪（PPP基准）为4万3874美元（约4699万韩元），位列新加坡（4万5755美元）、美国（4万4917美元）之后，排名第三位。紧随其后的分别是日本（4万3775美元）、德国（4万2254美元）、瑞士（3万9326美元）、荷兰（3万7218美元）等国。在调查对象国中，教师年薪最低的国家是埃及（1万604美元）。大部分国家的教师都表示，自己的年薪处于适当水平。韩国、日本、新加坡、美国的教师们则认为，自己现在的报酬处于合理水准以上。
有48%的韩国人表示&ldquo;会建议子女成为教师&rdquo;，位居中国（50%）之后，排名第2位。美国（33%）跃居上位圈，排名第7位。但日本（15%）却被挤到了第19位。类似回答也反映了教师的社会地位。实际上，在中国，教师的地位与&ldquo;医生&rdquo;相近。美国的&ldquo;图书馆管理员&rdquo;，日本的&ldquo;地方政府管理职位&rdquo;被分类为与教师拥有相似地位的职业群。
但韩国却只有11%的人表示&ldquo;学生尊敬教师&rdquo;，在这一调查项目中，排名垫底。与此相反，中国（75%）处于压倒性的第1位。紧随其后的依次为土耳其（52%）、新加坡（47%）。对于教育系统的信赖度，韩国得分仅为4.4分（满分10分），低于平均分数（5.5分），位列第19位。另外，对于&ldquo;教师学业履行的信赖度&rdquo;，韩国得分也比平均分数（6.3分）低，仅有5.4分。排名甚至低于埃及、捷克等国。该部门分数最高的国家为巴西（7.1分）。
报告书指出，&ldquo;就如同韩国的国际学生学业评价项目（PISA）排名较高一样，教师的地位也很高&rdquo;，&ldquo;但这种倾向并没有与各个领域都保持一致&rdquo;。&ldquo;虽然对于教育系统和教师的信赖度较低，但人们依旧建议子女成为教师&rdquo;。
