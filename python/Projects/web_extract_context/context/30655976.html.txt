亚太活力和中国外交阳关道一、 主题思想好，顺应时代潮流这次APEC领导人印尼巴厘岛峰会的主题是&ldquo;活力亚太，全球引擎&rdquo;，提倡在世界经济不景气的国际环境下，建立一个富有魅力的亚太，发挥世界经济增长的&ldquo;火车头&rdquo;作用。这是一个很好的主题思想，在当今国际形势下，APEC既需要&ldquo;定力&rdquo;，也需要进一步展现它特有的&ldquo;亲和力&rdquo;。它符合&ldquo;以合作共赢为核心的新型国际关系&rdquo;的时代诉求。目前，全球经济复苏仍步履艰难，存在诸多不确定因素，APEC成员普遍渴望同舟共济，共克时艰，发展更紧密的合作；探寻亚太地区新的&ldquo;定力&rdquo;和&ldquo;共同责任&rdquo;，发挥世界经济复苏和可持续发展的先锋作用。二、 APEC新的活力PK&ldquo;唱衰论&rdquo;。由于美国等老牌发达国家反复使用货币量化宽松政策和可能&ldquo;紧缩&rdquo;政策，其&ldquo;蝴蝶效应&rdquo;冲击着众多新兴经济体，印尼、中国以及印度等&ldquo;金砖国家&rdquo;，不同程度上都受到负面影响，增速放缓，国际上出现不少唱衰新兴经济体的论调。在这种情况下，APEC的绝大多数成员，特别是东盟、中国和韩国，看来将努力为&ldquo;火车头作用&rdquo;注入新的动力，实际上为&ldquo;唱衰论&rdquo;&ldquo;送葬&rdquo;。美国如能看清时代变迁和自己新的定位，也该有自知之明，为亚太、也为自己出一份力。三、实现&ldquo;茂物目标&rdquo;PK&ldquo;跨太平洋伙伴关系协议&rdquo;（TPP）。APEC历史上最大的战略成果之一就是1994年制定的茂物目标&mdash;&mdash;贸易和投资自由化与便利化，以及次年的大阪行动纲领，规定发达成员1010年、发展中成员2020年应&ldquo;全面和无条件地&rdquo;完成（如有困难时，允许有&ldquo;灵活性&rdquo;）。当时，美国为了有利于它的&ldquo;市场准入&rdquo;，咄咄逼人，先是要求&ldquo;达标&rdquo;路线图，后来又要搞什么&ldquo;早期收获&rdquo;，每年都要评估审核。现在已是1013年了，美国和日本等发达国家并未完成自己承担的义务，但却在紧锣密鼓地搞什么TPP,要超越茂物目标、甚至WTO，企图按照它的价值观和&ldquo;市场准入&rdquo;标准，占领新的&ldquo;制高点&rdquo;。这是不切实际的。这次会议强调要促进茂物目标的实现，加强互联互通，符合APEC战略目标。至于2020后怎么办，如何更上一层楼，如何更紧密地互联互通，包括美国热衷的TPP,都可以探讨。四、APEC &ldquo;大家庭精神&rdquo;和中国外交的阳关道。APEC诞生以来政治上最大的成果是，1993年领导人会议发表的西雅图宣言，庄严承诺要不断深化&ldquo;APEC大家庭精神&rdquo;，以及1996年苏比克会议通过的&ldquo;APEC方式&rdquo;，承认APEC存在巨大的多样性；允许灵活性和渐进性；遵循相互尊重、平等互利、协商一致、自主自愿原则；单边行动与集体行动相结合。1997年《温哥华领导人宣言》指出：&ldquo;APEC方式&rdquo;&ldquo;创造了国际经济合作的全新方式&rdquo;；很多专家学者认为，这是APEC大家庭的&ldquo;灵魂&rdquo;，具有强大的生命力。习近平首次出席APEC领导人会议，并访问印尼和马来西亚，提出一系列战略性建议和主张，特别是关于共建&ldquo;海上丝绸之路&rdquo;、携手建设中国&mdash;&mdash;东盟&ldquo;命运共同体&rdquo;，以及筹建亚洲基础设施投资银行，说到人们心坎上去了。人们普遍期盼，中国为APEC 注入新的血液，与各成员一起为APEC开创新局面。中国领导人一定会不负众望，中国的&ldquo;亲和力&rdquo;也是冷战思维和新版&ldquo;中国威胁论&rdquo;挡不住的，从中亚到亚太，中国在周边走的是共同的阳关道。日本如果一定要过它的&ldquo;独木桥&rdquo;，正像某些网友说的，随它去吧。实践和历史将会毫不留情。时代变迁量变进程正在不可阻挡地加速发展。明年中国将第二次担任东道主。可以预见，那将是更有意义的一次会议。
